Crystal Collector

Instructions


  1. You will be given a random number to get to. "Number To Get"
  2. There are four crystals below.
    • Blue
    • Green
    • Red
    • Orange
  3. Each crystal has a hidden value that when clicked will add to your total score.
  4. You win the game by matching your total score to the random number.
  5. You lose the game if your total score goes above the random number.
  6. Each time the game starts, the values of each crystal will change.
